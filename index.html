NY Stack Exchange

Github Live

NYC StackExchange is a clone of StackOverflow built on Rails and Backbone. Users can signup for accounts or sign in with Google, choose their profile pictures, and post NYC related Questions or Answers. The questions can contain supporting information including images or links.
All user/question/response date is accessed and stored using PostgreSQL.

Before the user logges in all the sites information is accessible via erb built static pages on the Rails server.
Once a user logs in, the whole app is rendered with Backbone, and updated with ajax calls to the rails server that then returns JSON.
App is made interactive with jQuery and custom CSS

  • Date: June 2015

jsSnake

Github Live

How to play: Snake starts automatically and can move pretty fast, change direction with the arrow keys. avoid hitting yourself and eat cherries and you will grow.

The "board" is a simple unordered list with a pseudo after clear-fix element and 256 line elements. The li's are floated left with a 1px border. Snake is written fully in JS and playable from node, The body segments are stored by an integer array, the snakes direction is also a single integer. The motion is rendered on screen by adding and removing classes on select line elements. The snake's tongue and eyes are :before and :after pseudo elements that are rotated and moved with relative position. The tongue is a simple 'Y' and the eyes are a colon with a white text-shadow; Snake moves forward ever 80ms.

  • Date: May 2015

pyChess

Github

A simple command line version of chess written in Python:
To play simply clone the repositiory from the github link above. You will automatically be askes if you would like to load a saved game, to save an active game type "save" or type "exit" to exit.
The game "board" is an object that keeps track of each piece positions in a one dimensional array, every piece is an object that inherits from a "prototype" piece based on shared ability.

  • Allows game saving and loading using Python's pickle
  • Keeps a game log in algebraic notation
